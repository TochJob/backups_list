# Project to implement UI and logic for backup

Стэк:

- VueJs v3 (Composition API)
- TypeScript
- CSS: TailWind
- Store: Pinia
- Bundler: Vite.js

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev -- for frontend
npm run serve:api -- for local db
```
