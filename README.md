# Project to implement UI and logic for backup

Stack:

- VueJs v3 (Composition API)
- TypeScript
- CSS: TailWind
- Store: Pinia
- Bundler: Vite.js

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

npm run dev -- star frontend application
npm run serve:api -- start local db

```sh
npm run dev
npm run serve:api
```
